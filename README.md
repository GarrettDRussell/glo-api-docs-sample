# GLO Industries — Event Stream API Documentation (Sample)

This repository contains a complete, fictional API documentation set for **GLO Industries**, a platform that provides real‑time event ingestion, processing, and subscription delivery.

The goal of this sample is to demonstrate professional, developer‑focused documentation using a docs‑as‑code workflow.

## Contents

- **Overview** — Platform architecture and core concepts  
- **Authentication** — API key and OAuth 2.0 flows  
- **Quickstart** — Send your first event  
- **API Reference** — Endpoints for events, sources, and subscriptions  
- **Integration Guides** — Step‑by‑step workflows  
- **Style Guide** — Writing standards and API conventions  
- **Diagrams** — Mermaid-based architecture and workflow diagrams  

All content is fictional and safe for public portfolios.
